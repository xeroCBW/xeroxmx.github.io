php基础语法

  1. 变量

    • php 是区分大小写的
    • 不能以数字开头

  2. 基本算数运算:与 c 基本相同

    +,-,*, /,%

  3. 比较运算符:

    >=,<,<=,==,!=,===,!==

    == 只验证值是否相等

    ===要求变量类型相等,且值也要相等;

  4. 逻辑运算符:

    基本运算:&&(并且) ||(或者)—与 c 一致

  5. 字符串拼接:

    字符串并不可以用逗号拼接的

    大串字符串拼接:heredoc/nowdoc

    更多字符串的用法,等用到的时候再去了解;

  6. 数组:

    数组不同于我们概念中的数组,他的数组有点像字典,里面都是有 key 的;有自己定义的 key; 或者是0,1,2等对应的字符串

  8. 函数:和 c 函数定义不一样,不需要写返回值;
