HTML 学习笔记

  1. 文件的相对位置

    • ./在本层文件夹中
    • ../在上层文件夹中

  2. 使用文字图标

    • 2.1 下载到本地,然后导入 fonts 文件夹,通过为对象进行设置

      css 和标签中写法不一( 字符的unicode码 http://fortawesome.github.io/Font-Awesome/cheatsheet/ )
      标签中是: &#xf042
      css中是: \f042

    • 2.2 通过网页链接来使用

      链接:<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.4.0/css/font-awesome.min.css">
      图标网址:(http://fontawesome.io/icons/)

  3. 色彩的表示:

    • black;white;red;green
    • 英文单词 #34F21D;
    • 十六进制 rgb(34,21,67);

  4. 定位:

    • 相对定位:相对自己的定位

    • 绝对定位:相对于 body,它会像PS的图层一样单独做一层，至于第几层你可以通过z-index:这个属性来设置

    • 改变 z-index: 2;来改变显示最外面

  5. overflow 样式

    • hidden,scroll[始终预留滚动条],auto[溢出时才有滚动条]
    • 默认是 visible:文本会超出显示区域显示

  7. textarea是需要写 id 的,否者会爆红

    <textarea name="introduction" id=" tesy" cols="30" rows="10"></textarea>

  9. zend studio 提醒错误,在 problem 时候会提醒
