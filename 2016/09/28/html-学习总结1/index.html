html 学习总结1

发表于 2016-09-28   |  

知识要点

  1. 锚点

    可以通过 a 标签定位到某一个地方:
<a href="./excise21.html#test">跳转到 excise17--id=main的位置</a>

  2. css 伪类:可以定义不同情况下的颜色

    伪类: 选中的元素的某个状态<br>
a:link 正常的a标签的状态<br>
a:visited 选中的是 鼠标点击后的a标签的状态<br>
a:hover 选中的是 鼠标经过a标签时的状态<br>
a:active 选中的是 鼠标点击瞬间(还未松鼠标)的状态<br>

  3. title 使用,用于鼠标选中时候的文字显示

  4. alt 使用, see 防盗链,这个不太清除
  5. 标签选中后不进行跳转可以设置 href=‘#’
  6. H5新特新: header footer aside(article)
  7. css 可以另外写一个文件
  8. css 选中可以用 id (唯一,#),标签(p div 等),类(class—.main),可以在标签中再去选(div ,main)
  9. table使用;th标题 td行 tr (row)列;合并样式可以去查询
  10. 块标签和行内标签相互转换: display 属性设置block;none
  11. 盒子模型,确定宽高,就会往里面进行压缩,盒子的大小不变
  12. 画三角形,使用 border,同时宽高设置为0
  13. html 一开始清除样式,可以参看京东个天猫的样式
  14. 块标签浮动,使用 float 类型
  15. ol 有序—会进行编号ul 无序–不会进行编号 li 是块标签
  16. 行内标签不可以舍hi在宽高,也不能设置 padding和 margin
  17. 块状标签:独占一行,可以设置标签,可以设置宽高

感悟

代码如果不给别人用,那代码就产生不了价值
